Quick Start

Validate an execution proposal:

curl -X POST https://api.pantheon.ai/ep/validate \
  -H "Content-Type: application/json" \
  -H "X-Agent-Key: your_key_here" \
  -d '{
    "proposal_id": "prop_001",
    "asset": "BNKR",
    "direction": "buy",
    "amount_usd": 100,
    "entry_price": 0.042,
    "stop_loss": 0.038,
    "take_profit": 0.05
  }'
