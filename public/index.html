API Reference

GET /ep/health (No auth)

Service health check.

GET /ep/status (No auth)

Service status and configuration.

GET /telemetry/status (No auth)

Live service health telemetry for dashboards.

GET /telemetry/validations (No auth)

Aggregated validation statistics (24h).

POST /ep/validate (X-Agent-Key required)

Validate proposal against policy. Returns verification result with proof_hash.

POST /ep/simulate (X-Agent-Key required)

Validate + simulate trade outcome. Returns expected PnL, drawdown, risk/reward.
