POST /ep/validate (X-Agent-Key required)

Validate proposal against policy. Returns verification result with proof_hash.

curl -X POST https://execution-protocol.onrender.com/ep/validate \
  -H "Content-Type: application/json" \
  -H "X-Agent-Key: $EP_KEY" \
  -d '{
    "proposal_id": "uuid",
    "asset": "BNKR",
    "direction": "buy",
    "amount_usd": 15,
    "entry_price": 0.042,
    "stop_loss": 0.038,
    "take_profit": 0.048,
    "leverage": 1,
    "policy_set_id": "olympus-v1"
  }'
