Integration Guide

Step-by-step guide to integrating Execution Protocol into your AI agent or application.

Quick Start

1

Install Dependencies

Choose your language and install the required packages:

npm install axios
pip install requests
# No installation needed - cURL is built into most systems
2

Configure Client

Set up your API client with the base configuration:

// JavaScript
const ExecutionProtocol = {
  baseURL: 'https://achillesalpha.onrender.com/ep',
  apiKey: 'your-api-key', // If required
  
  async validate(agentId, opportunity) {
    const response = await fetch(`${this.baseURL}/api/v1/validate`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ agent_id: agentId, opportunity })
    });
    return response.json();
  }
};

// Python
import requests

class ExecutionProtocol:
    def __init__(self, api_key=None):
        self.base_url = 'https://achillesalpha.onrender.com/ep'
        self.api_key = api_key
    
    def validate(self, agent_id, opportunity):
        response = requests.post(
            f"{self.base_url}/api/v1/validate",
            json={"agent_id": agent_id, "opportunity": opportunity}
        )
        return response.json()
3

Validate Opportunity

Send your opportunity for validation before execution:

const opportunity = {
  type: "hyperliquid_funding",
  asset: "ETH",
  expected_return: 0.05,  // 5%
  confidence: 0.90,       // 90% confidence
  max_capital: 5000       // $5,000
};

const decision = await ExecutionProtocol.validate(
  "my-agent-001",
  opportunity
);

console.log(decision.status);     // "approved" or "rejected"
console.log(decision.fee.feeEth); // "0.010000"
4

Execute Trade

If approved, execute the trade with the approval token:

if (decision.status === "approved") {
  const nonce = generateNonce(); // Your nonce generation
  
  const execution = await fetch(
    `${ExecutionProtocol.baseURL}/api/v1/execute`,
    {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        decision_id: decision.decision_id,
        approval_token: `APPROVE:${decision.decision_id}:EXEC:${nonce}`
      })
    }
  );
  
  const result = await execution.json();
  console.log(result.tx_hash); // Transaction hash
}

Complete Example

Trading Bot Integration

class TradingBot {
  constructor(agentId) {
    this.agentId = agentId;
    this.baseURL = 'https://achillesalpha.onrender.com/ep';
  }

  async analyzeAndTrade(marketData) {
    // 1. Analyze opportunity
    const analysis = this.analyze(marketData);
    
    // 2. Validate with Execution Protocol
    const decision = await this.validateOpportunity(analysis);
    
    if (decision.status !== "approved") {
      console.log("Trade rejected:", decision.reasoning);
      return null;
    }
    
    // 3. Check if fee is acceptable
    const feeUsd = parseFloat(decision.fee.feeUsd);
    const expectedProfit = analysis.expected_return * analysis.max_capital;
    
    if (feeUsd > expectedProfit * 0.1) { // Max 10% of profit
      console.log("Fee too high relative to profit");
      return null;
    }
    
    // 4. Execute trade
    const execution = await this.executeTrade(decision);
    
    // 5. Log result
    console.log("Trade executed:", execution.tx_hash);
    return execution;
  }

  async validateOpportunity(analysis) {
    const response = await fetch(`${this.baseURL}/api/v1/validate`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        agent_id: this.agentId,
        opportunity: analysis
      })
    });
    return response.json();
  }

  async executeTrade(decision) {
    const nonce = Date.now().toString();
    
    const response = await fetch(`${this.baseURL}/api/v1/execute`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        decision_id: decision.decision_id,
        approval_token: `APPROVE:${decision.decision_id}:EXEC:${nonce}`
      })
    });
    return response.json();
  }
}

Best Practices

1. Always Validate First

Never execute without validation. The validation step provides:

  • Risk assessment
  • Fee calculation
  • Maximum allocation limits
  • Onchain attestation

2. Handle Rejection Gracefully

if (decision.status === "rejected") {
  console.log("Reason:", decision.reasoning);
  // Adjust strategy based on rejection reason
  // e.g., increase confidence threshold, reduce allocation
}

3. Check Reputation

const reputation = await fetch(
  `${baseURL}/api/v1/reputation/${agentAddress}`
).then(r => r.json());

if (reputation.reputation_score < 5000) {
  console.log("Low reputation - reducing trade size");
  opportunity.max_capital *= 0.5;
}

4. Fee Management

Fee Structure: max(0.5%, 0.01 ETH)

  • Small trades (<$5,000): Flat 0.01 ETH (~$25)
  • Large trades (>$5,000): 0.5% of value

Error Handling

async function safeExecute(agentId, opportunity) {
  try {
    const decision = await validate(agentId, opportunity);
    
    if (decision.status === "approved") {
      return await execute(decision);
    }
    
    return { success: false, reason: decision.reasoning };
    
  } catch (error) {
    console.error("Execution Protocol Error:", error);
    
    if (error.message.includes("Rate limit")) {
      await sleep(60000); // Wait 1 minute
      return safeExecute(agentId, opportunity);
    }
    
    return { success: false, error: error.message };
  }
}

Rate Limiting

The API has a rate limit of 100 requests per minute. Implement exponential backoff for retries:

async function withRetry(fn, maxRetries = 3) {
  for (let i = 0; i < maxRetries; i++) {
    try {
      return await fn();
    } catch (error) {
      if (error.status === 429 && i < maxRetries - 1) {
        const delay = Math.pow(2, i) * 1000; // Exponential backoff
        await sleep(delay);
        continue;
      }
      throw error;
    }
  }
}
