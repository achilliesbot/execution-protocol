POST /api/v1/validate

Validate a trading opportunity and receive an execution decision.

Request Body

ParameterTypeRequiredDescription
agent_idstringYesUnique identifier for the agent
opportunity.typestringYesType: hyperliquid_funding, polymarket, dex_arbitrage
opportunity.assetstringYesAsset symbol (ETH, BTC, etc.)
opportunity.expected_returnnumberYesExpected return as decimal (0.05 = 5%)
opportunity.confidencenumberYesConfidence score 0.0-1.0
opportunity.max_capitalnumberYesMaximum capital in USD

Example Request

curl -X POST https://achillesalpha.onrender.com/ep/api/v1/validate \
  -H "Content-Type: application/json" \
  -d '{
    "agent_id": "dexter-research-001",
    "opportunity": {
      "type": "hyperliquid_funding",
      "asset": "ETH",
      "expected_return": 0.05,
      "confidence": 0.90,
      "max_capital": 5000
    }
  }'

Response 200 OK

{
  "decision_id": "d-1772954413261-dacmr1u5r",
  "status": "approved",
  "confidence_score": 0.9,
  "risk_level": "low",
  "max_allocation": 5000,
  "reasoning": ["High confidence score (>= 0.85)"],
  "fee": {
    "feeUsd": "25.00",
    "feeEth": "0.010000",
    "basisPoints": 50,
    "method": "percentage"
  },
  "attestation_uid": null,
  "timestamp": "2026-03-08T07:20:13.274Z"
}
POST /api/v1/execute

Execute an approved trade with fee collection.

Request Body

ParameterTypeRequiredDescription
decision_idstringYesDecision ID from validate endpoint
approval_tokenstringYesFormat: APPROVE:{decision_id}:EXEC:{nonce}

Example Request

curl -X POST https://achillesalpha.onrender.com/ep/api/v1/execute \
  -H "Content-Type: application/json" \
  -d '{
    "decision_id": "d-1772954413261-dacmr1u5r",
    "approval_token": "APPROVE:d-1772954413261-dacmr1u5r:EXEC:abc123"
  }'

Response 200 OK

{
  "decision_id": "d-1772954413261-dacmr1u5r",
  "status": "executed",
  "tx_hash": "0xe0bd03611de4...",
  "execution_price": 3200.50,
  "fee_paid": {
    "feeUsd": "25.00",
    "feeEth": "0.010000"
  },
  "timestamp": "2026-03-08T07:20:13.274Z"
}
